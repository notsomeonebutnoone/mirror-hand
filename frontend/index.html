System Overview

Signal capture, processing, and mechanical mirroring in one continuous loop.

Sensing Layer

Flex sensors and IMU streams capture finger curvature and wrist orientation at 120 Hz. The data is serialized and pushed over MQTT for immediate processing.

Python Signal Core

A Python-based filter stabilizes raw values, smoothing noise while maintaining responsiveness. Calibrated data is forwarded to the motion controller in under 5 ms.

Mechanical Mirror

Servo-driven linkages recreate the hand’s pose in real time. A proportional control layer keeps the mechanical hand aligned with the user’s gestures.

Signal Pipeline

A resilient MQTT backbone keeps latency low and message order consistent, even under noisy sensor conditions.

  • Capture sensor packets on embedded MCU
  • Stream to broker using QoS 1
  • Filter + normalize in Python
  • Publish motion targets to controller
